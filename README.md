# Derivatives project

### This project take an equation like `string` as input and return the derivative

#### INPUT FORMAT:

```
        ej1: 3x^2+5x+2 -> without  parentheses
        ej2: (3x^2+5x+2)*(2^x) -> with parentheses
        ej3: (3x^2+5x+2)*(x^2+2)
```

Supported derivative operations:
`exponential`
`chain rule`
`multiplication`
`summ & diff`
`exponentiation`
